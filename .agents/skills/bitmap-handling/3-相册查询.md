# 相册查询 (query)

## 概述

基于 MediaStore 的相册和媒体文件查询模块，所有查询在 IO 线程执行。

## 数据模型

### Album 相册实体

```kotlin
data class Album(
    val bucketId: Long = -1,      // 相册桶 ID
    val name: String?,             // 相册名称
    val coverUri: Uri?,            // 封面 Uri
    var count: Int = 0,            // 媒体文件数量
    val isCamera: Boolean = false  // 是否系统相机相册
)
```

### MediaFile 媒体文件

```kotlin
data class MediaFile(
    val id: Long,              // 媒体库唯一 ID
    val name: String?,         // 文件名
    val mimeType: String?,     // MIME 类型
    val size: Long,            // 文件大小（字节）
    val duration: Long,        // 时长（视频：毫秒；图片：0）
    val modifyTime: Long,      // 修改时间戳（秒）
    val bucketId: Long         // 所属相册 ID
)

// 扩展属性：获取 ContentUri
val MediaFile.contentUri: Uri
```

### AudioFile 音频文件

```kotlin
data class AudioFile(
    val id: Long,
    val name: String?,
    val mimeType: String?,
    val artist: String?,       // 演唱者
    val album: String?,        // 所属专辑
    val duration: Long,
    val size: Long,
    val modifyTime: Long
)
```

## MediaQueryManager 查询管理器

### 1. queryAllAlbums() - 查询所有相册

```kotlin
suspend fun queryAllAlbums(): List<Album>
```

**实现细节：**
- 查询 `MediaStore.Files` 获取图片 + 视频
- 按 `BUCKET_ID` 分组统计
- 按 `DATE_MODIFIED DESC` 排序，最新文件做封面
- 自动识别"Camera"相册

**投影列：**
```kotlin
arrayOf(
    BUCKET_ID,
    BUCKET_DISPLAY_NAME,
    _ID,              // 封面 ID
    MEDIA_TYPE,       // 区分图片/视频
    DATE_MODIFIED
)
```

### 2. queryAllImageAndVideo() - 查询所有媒体

```kotlin
suspend fun queryAllImageAndVideo(album: Album? = null): List<MediaFile>
```

**特点：**
- 支持指定相册过滤
- 按修改时间逆序排列
- 图片视频混合查询

**投影列：**
```kotlin
arrayOf(
    _ID, DISPLAY_NAME, MIME_TYPE, SIZE,
    DATE_MODIFIED, BUCKET_ID, DURATION
)
```

### 3. queryAllImages() - 查询所有图片

```kotlin
suspend fun queryAllImages(album: Album? = null): List<MediaFile>
```

**特点：**
- 支持指定相册过滤
- 按修改时间逆序排列
- 仅查询图片

**投影列：**
```kotlin
arrayOf(
    _ID, DISPLAY_NAME, MIME_TYPE, SIZE,

```

### 4. queryAllVideos() - 查询所有视频

```kotlin
suspend fun queryAllVideos(album: Album? = null): List<MediaFile>
```

**特点：**
- 支持指定相册过滤

**投影列：**
```kotlin
arrayOf(
    _ID, DISPLAY_NAME, MIME_TYPE, SIZE,
```

### 5. queryAllAudios() - 查询音频

```kotlin
suspend fun queryAllAudios(): List<AudioFile>
```

## 查询流程

```
MediaQueryManager(context)
    ↓
queryAllAlbums()
    ↓
1. 构建 Projection
2. 设置 Selection (媒体类型过滤)
3. 设置 SortOrder (时间逆序)
4. ContentResolver.query()
5. Cursor 遍历分组
6. 生成 Album 列表
```

## 使用示例

```kotlin
// 创建查询管理器
val queryManager = MediaQueryManager(context)

// 查询所有相册
val albums = queryManager.queryAllAlbums()
albums.forEach { album ->
    println("${album.name}: ${album.count} files")
    println("Cover: ${album.coverUri}")
}

// 查询指定相册的媒体
val cameraAlbum = albums.find { it.isCamera }
val mediaFiles = queryManager.queryAllImageAndVideo(cameraAlbum)

// 查询所有媒体（不区分相册）
val allMedia = queryManager.queryAllImageAndVideo()

// 查询音频
val audioFiles = queryManager.queryAllAudios()

// 仅查询图片
val images = queryManager.queryAllImages()

// 仅查询视频
val videos = queryManager.queryAllVideos()

// 转换为 ContentUri
mediaFiles.forEach { file ->
    val uri = file.contentUri
    // 用于加载或分享
}
```

## 性能优化

1. **协程 IO 线程** - 所有查询自动在 `Dispatchers.IO` 执行
2. **Cursor 分组** - 单次查询完成相册统计
3. **时间逆序** - 首个文件即为最新封面
4. **按需查询** - 先查相册列表，再查具体媒体

## 注意事项

1. **权限要求** - 需要 READ_EXTERNAL_STORAGE 或 READ_MEDIA_IMAGES
2. **Android 10+** - 使用 Scoped Storage 规则
3. **Cursor 关闭** - 使用 `use {}` 自动管理
4. **空值处理** - 文件名、MIME 类型可能为 null
