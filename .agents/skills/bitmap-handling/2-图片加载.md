# 图片加载 (loader)

## 概述

封装 ImageDecoder 的图片加载模块，支持原图、压缩、缩略图三种加载模式。

## 核心配置 ImageLoadConfig

```kotlin
data class ImageLoadConfig(
    val maxWidth: Int = 1440,          // 最大宽度
    val maxHeight: Int = 1920,         // 最大高度
    val quality: Quality = MEMORY_POLICY_DEFAULT,  // 内存策略
    val ignoreFileTypes: List<String> = ["gif", "webp", "svg"],
    val ignoreSizeInKB: Int = 1200,    // 忽略阈值
    val alwaysLoadOriginal: Boolean = false,  // 总是加载原图
    val targetWidth: Int = 0,          // 固定目标宽度
    val targetHeight: Int = 0          // 固定目标高度
)
```

**Quality 枚举：**
- `MEMORY_POLICY_DEFAULT` - 默认内存策略
- `MEMORY_POLICY_LOW_RAM` - 低内存模式

## 三大加载函数

### 1. loadOriginalUriOrFile - 加载原图

```kotlin
suspend fun loadOriginalUriOrFile(
    context: Context,
    uriOrFile: Any,  // Uri 或 File
    config: ImageLoadConfig = ImageLoadConfig(
        quality = MEMORY_POLICY_DEFAULT,
        alwaysLoadOriginal = true
    )
): Bitmap?
```

**特点：**
- 忽略所有压缩配置
- 使用 `ALLOCATOR_SOFTWARE`
- 适合需要完整图片信息的场景

### 2. loadCompressUriOrFile - 压缩加载

```kotlin
suspend fun loadCompressUriOrFile(
    context: Context,
    uriOrFile: Any,
    config: ImageLoadConfig = ImageLoadConfig(
        maxWidth = 1440,
        maxHeight = 1920,
        quality = MEMORY_POLICY_LOW_RAM,
        ignoreSizeInKB = 2000
    )
): Bitmap?
```

**特点：**
- 自动采样和缩放
- 智能忽略小文件
- 适合大多数显示场景

### 3. loadThumbnailUriOrFile - 缩略图加载

```kotlin
suspend fun loadThumbnailUriOrFile(
    context: Context,
    uriOrFile: Any,
    thumbSize: Size?  // null 则降级为压缩加载
): Bitmap?
```

**特点：**
- 优先使用系统缩略图缓存
- 支持图片/视频
- 兼容 Android 8+

## ImageLoadHelper 内部实现

**加载流程：**
1. **解析 Uri** - 获取文件大小和格式
2. **判断忽略** - 检查是否需要压缩
3. **创建 ImageDecoder.Source** - File 或 ContentResolver
4. **配置解码器** - 设置采样率、目标尺寸
5. **解码 Bitmap** - `ImageDecoder.decodeBitmap()`

**采样计算：**
```kotlin
// 计算采样率
val heightRatio = origH / reqHeight
val widthRatio = origW / reqWidth
val inSample = minOf(heightRatio, widthRatio)

// 计算目标尺寸
val (targetW, targetH) = calculateTargetDimensions(
    origW, origH, maxWidth, maxHeight
)
```

## ThumbnailLoadHelper 缩略图工具

**核心方法：**

1. **createImageThumbnailByPath()** - 文件路径生成缩略图
   - API 29+ 使用 `ThumbnailUtils.createImageThumbnail()`
   - 优先查询系统缓存

2. **loadThumbnailCompat()** - 兼容加载（Android 8+）
   - 自动判断图片/视频
   - 统一接口处理 ContentUri/FileUri

3. **getVideoThumbnailByRetriever()** - 视频缩略图
   - 使用 `MediaMetadataRetriever`
   - 支持任意时间点截图

## 使用示例

```kotlin
// 加载原图
val originalBitmap = loadOriginalUriOrFile(context, uri)

// 压缩加载（适合 ImageView）
val compressBitmap = loadCompressUriOrFile(
    context,
    uri,
    ImageLoadConfig(
        maxWidth = 1080,
        maxHeight = 1080,
        quality = ImageLoadConfig.Quality.MEMORY_POLICY_LOW_RAM
    )
)

// 缩略图加载（适合列表项）
val thumbnailSize = Size(240, 320)
val thumbnailBitmap = loadThumbnailUriOrFile(context, uri, thumbnailSize)

// 视频缩略图
val videoThumbnail = ThumbnailLoadHelper(context)
    .getVideoThumbnailByRetriever(videoUri, thumbnailSize)
```

## 最佳实践

1. **复用实例** - `ThumbnailLoadHelper` 建议全局单例
2. **选择合适尺寸** - 使用系统常用尺寸（640x480 等）提高缓存命中率
3. **内存控制** - 列表场景使用 `MEMORY_POLICY_LOW_RAM`
4. **异常处理** - 所有加载函数返回 null 表示失败
