# 图片压缩 (compressor)

## 概述

图片压缩模块，支持智能质量选择、二次压缩、EXIF 旋转校正。

## 核心类

### BestImageCompressor

主压缩类，支持流式压缩。

**配置项 Config：**
- `maxWidth/maxHeight` - 最大尺寸（默认 1440x1920）
- `qualityType` - 质量策略：`default`/`deep`/`shallow`
- `secondReduce` - 二次压缩配置（目标大小、步长、尝试次数）
- `supportScale` - 是否启用缩放
- `ignoreFileTypes` - 忽略格式（gif/webp/svg）
- `ignoreSizeInKB` - 忽略阈值（默认 150KB）
- `alwaysCopyMode` - 失败回退策略（"all"：全部拷贝，"onlyUri"：仅远程Uri拷贝，null：不拷贝）

**二次压缩 SecondReduce：**
- `targetSize` - 目标字节数
- `qualityStep` - 质量步长（默认 5）
- `secondScaleRatio` - 二次缩放比例（默认 0.9）
- `tryReduceCount` - 尝试次数（默认 3）

### 压缩流程

1. **解析 Uri** - `myParse()` 获取文件信息
2. **判断忽略** - 检查大小和格式
3. **创建压缩器** - File/Uri 分别处理
4. **解码采样** - `ImageDecoder` + `inSampleSize`
5. **EXIF 旋转** - 自动校正方向
6. **压缩写入** - 按质量压缩为 JPEG
7. **二次压缩** - 超标时循环降质量
8. **缓存记录** - `CompressCacheConstManager`

### 工具函数

**useCompress(context, uri, config)** - 公开 API
- 自动判断是否忽略
- 失败回退到拷贝
- 返回压缩后的 File

**chooseQuality(size, qualityType)** - 智能质量选择
```kotlin
// deep 策略示例
size > 16MB → quality 55
size > 8MB  → quality 65
size > 4MB  → quality 78
```

**calculateSampleSize()** - 计算采样率
**calculateTargetDimensions()** - 计算目标尺寸

### 缓存管理

**CompressCacheConstManager**
- `createCompressOutputFile()` - 生成压缩输出文件名
- `createCopyOutputFile()` - 生成拷贝输出文件名
- `afterFileOperator()` - 记录 LRU 操作
- `cleanSpace()` - 清理缓存

## 使用示例

```kotlin
// 基础压缩
val compressed = useCompress(context, uri)

// 自定义配置
val config = BestImageCompressor.Config(
    maxWidth = 1920,
    maxHeight = 1080,
    qualityType = "deep",
    secondReduce = BestImageCompressor.SecondReduce(
        targetSize = 500 * 1024,  // 500KB
        qualityStep = 5,
        tryReduceCount = 3
    )
)
val compressed = useCompress(context, uri, config)

// 直接调用压缩器
val compressor = BestImageCompressor(
    config = config,
    provideInputStream = { context.contentResolver.openInputStream(uri) },
    provideFileSize = { parsedInfo.fileLength },
    provideImageDecodeSource = { ImageDecoder.createSource(context.contentResolver, uri) }
)
val result = compressor.compress()
```

## 注意事项

1. **必须在 IO 线程运行** - 使用 `suspend` 函数
2. **EXIF 自动处理** - 无需手动旋转
3. **缓存清理** - 定期调用 `cleanSpace()`
4. **失败回退** - 压缩失败自动拷贝原图
